朱砂在中国画中主要用在哪些场景？能帮我整理一个简短清单吗？

朱砂（Cinnabar, HgS）是中国传统矿物颜料里最经典的暖红色，在绘画里的用法大致集中在几个场景：

  • 人物画设色——唐宋人物画用朱砂晕染唇、颊，加一点胭脂可得"桃花面"。
  • 工笔花鸟——红果、红花，尤其是牡丹、石榴、寒梅，常用朱砂分染 + 淡朱罩染。
  • 青绿山水点苔——李思训派在石绿石青之上点朱砂苔，形成冷暖撞色节奏。
  • 题跋朱印——收藏印、压脚印几乎全部采用朱砂印泥，红白对比最夺目。
  • 道教符箓 & 门神——民俗绘画中朱砂具避邪寓意，常用朱笔点睛。

如果你要在界面设计里借鉴这个颜色：

:root {
  --cinnabar: #C2402C;     /* 朱砂主色 */
  --cinnabar-strong: #D95E3B;
  --seal-red: #8E2A1D;     /* 印章深红 */
}
.btn-primary {
  background: linear-gradient(135deg, var(--cinnabar), var(--seal-red));
  color: #fff8ef;
}

朱砂的视觉特征是高饱和 + 中明度，在浅底上最容易形成视觉焦点，但一定要控制用量——传统画论里讲"色不过三，朱砂尤贵"，铺得太多会压迫眼睛。
